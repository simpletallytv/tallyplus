	\section{Introduction}\label{about}
	The device is designed to work with system and portable cameras in television studios.
	\noindent Purpose: 
	\begin{itemize}
		\item Camera number identification
		\item On-Air tally
	\end{itemize}
	\noindent The On-Air tally is controlled by one of the following methods:
	\begin{itemize}
		\item LDR sensor (light-dependent resistor)
		\item GPI signal (General Purpose Interface)	
	\end{itemize}
	
	\section{Design}\label{design}
	The device features:
	\begin{itemize}
		\item Microprocessor control
		\item Indicator number configuration
		\item Brightness configuration
		\item Light sensor sensitivity configuration
		\item Non-volatile memory for storing settings
	\end{itemize}

	\section{Operating Procedure}
	\begin{enumerate} \sloppy
		\item Install and secure the indicator on the camera or teleprompter. Use photo accessories with 1/4” threaded mounting for installation.
		\item Connect the cables according to the chosen connection option.
		\item Use the toggles located on the right side of the indicator to set the number and brightness of the indicators. The manipulators are 3-position with a lock in the middle position .
		\item In the mode of operation with the LDR sensor (photoresistor), use the sensitivity regulator located on the side to set the threshold for switching the indicator color. When operating from the GPI signal, set the sensitivity regulator to the minimum position (turn counterclockwise until it clicks).
		\item Check the correct operation of the indicator by switching the camera to On-Air / Off-Air modes.
	\end{enumerate}
